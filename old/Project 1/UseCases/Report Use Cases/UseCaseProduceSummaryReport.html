Use case: Produce Summary Report extends Manage Reports

Context: The Produce Summary Report use case produces a summary of the week’s accounts payable.

Actors: Manager, Timer

Main Success Scenario:

  1. The software product produces a file containing the following information
  1. A list of every provider to be payed that week
  2. Number of consultations for each provider
  3. Total fee for each provider
  4. Number of providers who provided services that week
  5. Total number of consultations from all providers that week
  6. Overall fee
  1. The system saves the file.
