Use case: Produce Provider Report extends Manage Reports

Context: The Produce Provider Report use case produces a list of services that a provider administered in a given week.

Actors: Manager, Timer

Main Success Scenario:

  1. The system uses the provider number to find the provider’s records.
  2. The software product produces a file containing the following information:
  1. Provider Name
  2. Provider Number
  3. Provider City
  4. Provider State
  5. Provider ZIP code
  6. For every service
  1. Date of service
  2. Date and time that service report was received
  3. Member name
  4. Member number
  5. Service code
  6. Fee to be paid
  1. Total number of consultations (3 digits)
  2. Total fee for week (up to $99,999.99)
  1. The system saves the file.

Extensions:

      1a. The provider number is invalid.

        1a.1. The use case is terminated with an error.
