Use case: Produce Member Report extends Manage Reports

Context: The Produce Member Report use case produces a list of each service that a member received in a given week.

Actors: Manager, Timer

Main Success Scenario:

  1. The system uses the member number to find the member’s records.
  2. The system produces a file containing the following information:
  1. Member Name
  2. Member Number
  3. Member City
  4. Member State
  5. Member ZIP code
  6. For every service
  1. Date of service
  2. Provider name
  3. Service name
  1. The system saves the file.

Extensions:

      1a. The member number is invalid.

        1a.1. The use case is terminated with an error.
